# Changelog

All notable changes to the **Nexora IPTV Global** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [2.4.0] - 2026-07-13
### Added
- Created a dedicated **FIFA World Cup 2026 Match Center** navigation with auto-discovery tag indexers.
- Added dynamic Category Counters directly in the Sidebar view.
- Added premium **Help Center** and **About Nexora** visual modal controllers.
- Configured modular documentation, including Architecture, Security, Development, and Release Strategies.

### Changed
- Integrated responsive drawer layout for mobile streams, collapsing the full-size sidebar elegantly.
- Optimised M3U parser regular expressions for speed and extreme file sizes.

---

## [2.3.0] - 2026-04-10
### Added
- Introduced custom **HLS.js player control hub** featuring interactive progress slider, volume cache, theater size toggle, and video speed control.
- Integrated `motion/react` dynamic transition states on active sidebar tabs.

### Fixed
- Resolved video loading issues with CORS-protected stream proxies.

---

## [2.1.0] - 2025-11-05
### Added
- Implemented client-side **Local Storage State Persistence** for recently watched history, custom user playlists, and global theme configurations.
- Added Search Input focused triggers automatically linked to global stream databases.

---

## [1.0.0] - 2025-06-12
### Added
- Initial official stable release of Nexora stream aggregator.
- Core Extended M3U playlist file uploader and parser engine.
- Responsive bento-grid layout for channel listings.
